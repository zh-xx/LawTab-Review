# LawTab Review

## What it does
- Import TXT / PDF / DOCX contracts and validate size before review
- Generate a structured review flow: flowchart, overview, and risk tables (foundation/business/legal)
- Keep review history and resume AI conversations per contract
- Configure model provider (DeepSeek default or OpenAI-compatible endpoint)

## Build
1) Open `contract review.xcodeproj` in Xcode
2) Select target `LawTab Review`
3) Run

## Requirements
- macOS 26
